Spring Boot + Dubbo实践

概述

​ 使用Spring Boot + Dubbo + zookeeper搭建一个简单的分布式服务。

Dubbo是阿里巴巴开源的基于 Java 的高性能 RPC（一种远程调用） 分布式服务框架（SOA），致力于提供高性能和透明化的RPC远程服务调用方案，以及SOA服务治理方案。
